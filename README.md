# RAG - FastAPI

## Introduction
This repo creates a [RAG](https://github.com/jackpck/RAG) API using **FastAPI**.

## Instructions
- At the project root, run `python -m app.main`
- Enter http://localhost:8000/docs in the browser
- Under default, you will see three endpoints
    - `POST /chat`
    - `POST /home`
    - `GET /home{input}`
- Under `POST /chat`, click *Try it out*. Type in a query to the value of the dict and click *Execute*.
